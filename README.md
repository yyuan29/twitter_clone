# Twitter_clone

This is my version of Twitter. It is a simple Flask application that displays messages from a database by using the FastAPI web framework and sqlite3. 


## Features
- SQLite Integration
- Jinja2 Template Inheritance
- Static CSS and Image serving

## How to Run
1. Run `python db_create.py` to initialize the database.
2. Run `python main.py` to start the server.
3. Visit `http://127.0.0.1:8888`
